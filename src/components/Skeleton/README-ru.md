# Skeleton

<!--/GITHUB_BLOCK-->

```tsx
import {Skeleton} from '@gravity-ui/uikit';
```

Компонент `Skeleton` отображает временный макет контента, пока фактические данные загружаются. Такой предварительный макет используется для снижения раздражения, вызываемого ожиданием загрузки.

## Variant

Используйте свойство `variant` для выбора варианта отображения скелетона. Значение по умолчанию — `'rounded'`.

| Значение    | Описание                                                                                                                                                                                      |
| :---------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `'square'`  | Острые углы (`border-radius: 0`)                                                                                                                                                              |
| `'rounded'` | Небольшое скругление (`border-radius: 5px`) — **по умолчанию**                                                                                                                                |
| `'circle'`  | Круглая форма (`border-radius: 50%`)                                                                                                                                                          |
| `'text'`    | Наследует `font-size` и `line-height` из родительского контекста; Используется для многострочных текстовых заглушек - разместите несколько скелетонов внутри `<Text variant="...">` элемента. |

<!--GITHUB_BLOCK-->

```tsx
<Skeleton variant="circle" style={{width: 40, height: 40}} />

<Text variant="body-1">
    <Skeleton variant="text" style={{width: 200}} />
</Text>

<Text variant="body-1">
    <Skeleton variant="text" style={{width: 400}} />
    <Skeleton variant="text" style={{width: 400}} />
    <Skeleton variant="text" style={{width: 240}} />
</Text>
```

<!--/GITHUB_BLOCK-->

## Дочерние элементы (`children`)

Передайте один `React.ReactElement` в качестве `children`, чтобы использовать скелетон как оверлей, который автоматически наследует размеры обёрнутого элемента. Дочерний элемент рендерится внутри контейнера скелетона (с `aria-hidden="true"`).

<!--GITHUB_BLOCK-->

```tsx
<Skeleton>
    <Button size="m">Сохранить</Button>
</Skeleton>

<Skeleton variant="circle">
    <Avatar size="l" shape="circle" imgUrl={url} alt="Пользователь" aria-label="avatar" />
</Skeleton>
```

## Свойства

| Имя       | Описание                                                                                                                                                      |                         Тип                         | Значение по умолчанию |
| :-------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------ | :-------------------------------------------------: | :-------------------: |
| style     | Пользовательские CSS-стили корневого элемента.                                                                                                                |                `React.CSSProperties`                |                       |
| className | Пользовательский CSS-класс корневого элемента.                                                                                                                |                      `string`                       |                       |
| qa        | HTML-атрибут `data-qa`, используется для тестирования.                                                                                                        |                      `string`                       |                       |
| animation | Тип анимации, применяемый к скелетону.                                                                                                                        |        `'gradient'` \| `'pulse'` \| `'none'`        |      `gradient`       |
| variant   | Вариант отображения скелетона (`'text'` — строчная текстовая заглушка, наследующая типографику из контекста; остальные значения управляют скруглением углов). | `'square'` \| `'rounded'` \| `'circle'` \| `'text'` |       `rounded`       |
| children  | Элемент для обёртки; скелетон принимает его размер.                                                                                                           |                `React.ReactElement`                 |                       |
