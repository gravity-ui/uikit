# Skeleton

<!--/GITHUB_BLOCK-->

```tsx
import {Skeleton} from '@gravity-ui/uikit';
```

Компонент `Skeleton` отображает временный макет контента, пока фактические данные загружаются. Такой предварительный макет используется для снижения раздражения, вызываемого ожиданием загрузки.

## Анимация

Используйте свойство `animation` для управления анимацией загрузки. Значение по умолчанию — `'gradient'`.

| Значение     | Описание                                        |
| :----------- | :---------------------------------------------- |
| `'gradient'` | Скользящий градиентный блеск — **по умолчанию** |
| `'pulse'`    | Пульсация прозрачности                          |
| `'none'`     | Без анимации                                    |

## Форма

Используйте свойство `shape` для управления border-radius скелетона. Значение по умолчанию — `'rounded'`.

| Значение    | Описание                                                                           |
| :---------- | :--------------------------------------------------------------------------------- |
| `'rounded'` | border-radius с токеном, зависящим от размера — **по умолчанию**                   |
| `'sharp'`   | Без border-radius (`border-radius: 0`)                                             |
| `'circle'`  | Круглая форма (`border-radius: 50%`, `aspect-ratio: 1`)                            |
| `'square'`  | Квадратные пропорции (`aspect-ratio: 1`) с тем же border-radius, что у `'rounded'` |

## Размер

Используйте свойство `size` для задания высоты скелетона (а также ширины для форм `'circle'` и `'square'`). Значение по умолчанию — `'m'`. Каждый размер также влияет на токен border-radius для форм `'rounded'` и `'square'`.

| Значение | Описание                   |
| :------- | :------------------------- |
| `'xs'`   | Очень маленький            |
| `'s'`    | Маленький                  |
| `'m'`    | Средний — **по умолчанию** |
| `'l'`    | Большой                    |
| `'xl'`   | Очень большой              |

<!--GITHUB_BLOCK-->

```tsx
<Skeleton size="xs" style={{width: 120}} />
<Skeleton size="m"  style={{width: 200}} />
<Skeleton size="xl" style={{width: 280}} />

<Skeleton shape="circle" size="l" />
<Skeleton shape="square" size="l" />
```

<!--/GITHUB_BLOCK-->

## Текстовая заглушка

Установите `isText` в `true`, чтобы скелетон наследовал `font-size` и `line-height` из родительского контекста. Разместите несколько скелетонов внутри `<Text variant="...">` для многострочных заглушек.

<!--GITHUB_BLOCK-->

```tsx
<Text variant="body-1">
  <Skeleton isText style={{width: 400}} />
  <Skeleton isText style={{width: 400}} />
  <Skeleton isText style={{width: 240}} />
</Text>
```

## Свойства

| Имя         | Описание                                                                                                                                   |                         Тип                          | Значение по умолчанию |
| :---------- | :----------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------: | :-------------------: |
| `style`     | Пользовательские CSS-стили корневого элемента.                                                                                             |                `React.CSSProperties`                 |                       |
| `className` | Пользовательский CSS-класс корневого элемента.                                                                                             |                       `string`                       |                       |
| `qa`        | HTML-атрибут `data-qa`, используется для тестирования.                                                                                     |                       `string`                       |                       |
| `animation` | Тип анимации, применяемый к скелетону.                                                                                                     |        `'gradient'` \| `'pulse'` \| `'none'`         |      `gradient`       |
| `shape`     | Вариант border-radius. `'circle'` и `'square'` также задают `aspect-ratio: 1`; `'sharp'` убирает border-radius полностью.                  | `'rounded'` \| `'sharp'` \| `'square'` \| `'circle'` |       `rounded`       |
| `size`      | Предустановленная высота (и ширина для форм `'circle'`/`'square'`). Также управляет токеном border-radius для форм `'rounded'`/`'square'`. |     `'xs'` \| `'s'` \| `'m'` \| `'l'` \| `'xl'`      |          `m`          |
| `isText`    | Если `true`, высота определяется из `font-size` и `line-height` родителя. Используется для строчных текстовых заглушек.                    |                      `boolean`                       |                       |
