# Skeleton

<!--/GITHUB_BLOCK-->

```tsx
import {Skeleton} from '@gravity-ui/uikit';
```

Семейство компонентов `Skeleton` отображает временный макет контента, пока фактические данные загружаются. Такой предварительный макет используется для снижения раздражения, вызываемого ожиданием загрузки. В семейство входит базовый примитив `Skeleton` и несколько готовых вариантов, повторяющих форму реальных UI-компонентов.

## Компоненты

| Компонент           | Описание                                                 |
| :------------------ | :------------------------------------------------------- |
| `Skeleton`          | Базовый прямоугольный блок-скелетон                      |
| `SkeletonAvatar`    | Заглушка аватара (круглая или квадратная)                |
| `SkeletonButton`    | Заглушка кнопки с настраиваемой шириной                  |
| `SkeletonLabel`     | Заглушка метки/бейджа                                    |
| `SkeletonText`      | Заглушка текстового блока (одна или несколько строк)     |
| `SkeletonUser`      | Заглушка карточки пользователя (аватар + имя + описание) |
| `SkeletonUserLabel` | Заглушка пользовательской метки с настраиваемой шириной  |

## Использование

```tsx
import {
  Skeleton,
  SkeletonAvatar,
  SkeletonButton,
  SkeletonLabel,
  SkeletonText,
  SkeletonUser,
  SkeletonUserLabel,
} from '@gravity-ui/uikit';

// Базовый блок-скелетон
<Skeleton style={{height: 30, width: 200}} />

// Заглушка аватара
<SkeletonAvatar size="m" shape="circle" />

// Многострочная заглушка текста
<SkeletonText variant="body-1" lines={3} lastLineWidth="60%" />

// Заглушка карточки пользователя
<SkeletonUser size="m" />
```

## `Skeleton`

Базовый примитив. Используйте его для любой заглушки произвольной формы, управляя размером через `style` или `className`.

### Свойства

| Имя       | Описание                                               |                  Тип                  | Значение по умолчанию |
| :-------- | :----------------------------------------------------- | :-----------------------------------: | :-------------------: |
| animation | Тип анимации, применяемый к скелетону.                 | `'gradient'` \| `'pulse'` \| `'none'` |     `'gradient'`      |
| className | Пользовательский CSS-класс корневого элемента.         |               `string`                |                       |
| style     | Пользовательские CSS-стили корневого элемента.         |         `React.CSSProperties`         |                       |
| qa        | HTML-атрибут `data-qa`, используется для тестирования. |               `string`                |                       |

## `SkeletonText`

Скелетон в форме текстового блока. Поддерживает однострочный и многострочный режимы.

```tsx
import {SkeletonText} from '@gravity-ui/uikit';

// Одна строка
<SkeletonText variant="body-1" width="200px" />

// Несколько строк
<SkeletonText variant="body-1" lines={3} lastLineWidth="60%" />
```

### Свойства

| Имя           | Описание                                                                                |                  Тип                  | Значение по умолчанию |
| :------------ | :-------------------------------------------------------------------------------------- | :-----------------------------------: | :-------------------: |
| animation     | Тип анимации.                                                                           | `'gradient'` \| `'pulse'` \| `'none'` |     `'gradient'`      |
| className     | Пользовательский CSS-класс корневого элемента.                                          |               `string`                |                       |
| lastLineWidth | Ширина последней строки при `lines > 1`. Если не задано, используется значение `width`. |         `number` \| `string`          |                       |
| lines         | Количество строк текста.                                                                |               `number`                |          `1`          |
| variant       | Вариант текста (определяет высоту строки).                                              |               `string`                |      `'body-1'`       |
| width         | Ширина каждой строки.                                                                   |         `number` \| `string`          |       `'100%'`        |
