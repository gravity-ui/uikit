## Skeleton

Компонент для иллюзии быстрой загрузки страниц.

### PropTypes

| Property  | Type     | Required | Default | Description                                                  |
| :-------- | :------- | :------- | :------ | :----------------------------------------------------------- |
| className | `String` |          |         | Кастомный класс, используется для задания размеров скелетона |
