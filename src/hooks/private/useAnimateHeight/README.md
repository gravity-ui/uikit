# useAnimateHeight

The `useAnimateHeight` hook handles dynamic height change transition

## Properties

| Name | Description                     |       Type        | Default |
| :--- | :------------------------------ | :---------------: | :-----: |
| ref  | Ref-link to target of animation | `React.RefObject` |         |

## Result

```
[
    onTransitionEnd, // () => void
]
```
