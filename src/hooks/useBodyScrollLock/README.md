# useBodyScrollLock

<!--/GITHUB_BLOCK-->

```tsx
import {useBodyScrollLock} from '@gravity-ui/uikit';
```

The `useBodyScrollLock` hook helps to blocks scrolling on the body element.

## Properties

| Name    | Description |   Type    | Default |
| :------ | :---------- | :-------: | :-----: |
| enabled | Enable flag | `boolean` |         |
